# Odin Frontend

Requires Node `v4.1.x`+ and NPM `v2.14.x`+.

Originally built with Node `v8.11.3` and NPM `v5.6.0`.

## Setup

```
npm install
```

## Run Dev server

```
npm start
```

## Single build

```
npm run build
```
